Place your VRM model file here as:

  apps/web/public/model.vrm

The scene will automatically load it on startup.
If no model is found, a placeholder cube will be shown as fallback.

You can download sample VRM files from:
  https://hub.vroid.com  (free models)
  https://github.com/pixiv/three-vrm/blob/dev/packages/three-vrm/tests/models/
